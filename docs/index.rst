libfranka client library
========================

This documentation should help you to get started with libfranka. For a quick check on compatibility with the Robot
System Version, head over to :ref:`compatibility-libfranka`.

libfranka is a C++ library that allows you to control a Franka robot from a remote computer. The library is open-source and can be found on
`GitHub <https://github.com/frankarobotics/libfranka>`_.
